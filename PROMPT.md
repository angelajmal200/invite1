# Prompt used for this parallax wedding website

Create a premium smooth-scroll Kerala wedding parallax website using the provided transparent PNG assets.

Scroll sequence:
1. Temple gopuram appears slowly from the background.
2. Banana leaves slide in from the left and right.
3. Brass hanging lamps fade in softly.
4. Jasmine garlands descend from the top.
5. Pink lotus flowers float upward from the bottom corners.
6. The wedding couple rises smoothly from the bottom.
7. The couple stops at center as the final hero focus.
8. “Ajmal & Bride” fades in only after the couple settles.
9. Scroll continues into a wedding details section.

Design requirements:
- Use the created PNG assets as independent parallax layers.
- No large blurred card blocking the background or couple.
- Mobile-first responsive layout.
- Smooth lerped scroll animation using JavaScript requestAnimationFrame.
- Luxury Kerala wedding invitation aesthetic.
- Warm cream, pastel pink, ivory, gold and turquoise palette.
- Professional website structure with index.html, style.css, script.js, assets folder.
- GitHub Pages compatible.
